# Prototype

`loom-prototype-v2.jsx` lives in the source Claude project and is too large
to copy through this Claude session in one shot. It is **not blocking until
W11** (SwiftUI build); v1 W1–W10 do not reference it.

To populate this folder when needed:

1. Open the Claude project containing the design corpus.
2. Download `loom-prototype-v2.jsx` directly, or have a future Claude Code
   session paste it here in chunks.
3. Drop the file at `loom-meta/docs/prototype/loom-prototype-v2.jsx`.

The style guide (`loom-meta/docs/loom-style-guide.md`) is the authoritative
visual reference until then.
